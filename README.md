# 돌려돌려 돌림판

## Overview

CSS 를 응용한 간단한 로직으로 구현한 돌림판

## Feature

- 오직 CSS 와 약간의 JS 만을 사용해서 구현
- 위치 값 계산해서 뭐가 당첨되었는지 알려주는 코드 추가
- 선택지 8개에서 10개로 증가
